APO-DZ

Applications to facilitate community service and communication for my undergrad community service group, Alpha Phi Omega. The app provides native interfaces on iOS and Android to communicate with fellow brothers and to create/search/sign up for/report service projects & other chapter events.

Mater Dolorosa App

Native Android app for my parish. Currently, it uses the current date and time to provide up-to-date information about service times and office hours, as well as easing access to appropriate personnel. Future plans involve creating a dashboard for current and future events, but that entails setting up a backend somewhere and more careful planning.

  • Repo on GitHub: https://github.com/ronaldsmartin/mater-dolorosa-android

Penn Philippine Association App

Following a lot of interest in the APO app's directory functionality from non-APO classmates, by the end of this summer I hope to generalize the patterns implicit in APO-DZ and build a generic directory module that college student organizations can adapt. The first fork shall be for my favorite collegiate Filipinos, the Penn Philippine Association.

Automated Theorem Deducers

As a big fan of recursion and functional programming, one of my interests has been working on modeling proof systems in Haskell. The original version for Gentzen's sequent calculus was built during Penn's FA'13 Advanced Programming course with my partner, Anne Foster. Code will be open-sourced as time permits.
