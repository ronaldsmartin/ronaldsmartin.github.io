Current Projects

APO-DZ

Applications to facilitate community service and communication for my undergraduate community service group, Alpha Phi Omega. The app provides native interfaces on iOS and Android to communicate with fellow brothers and to create/search/sign up for/report service projects & other chapter events.

As an app to help my friends and the local Philadelphia community, APO-DZ has served as my primary point of entry for exploratory forays into different platforms:

  • the Android version began and remains here on GitHub
  • the iOS version is currently hiding in a private repo until I can remove all private data
  • a web version built using Google Polymer web components is in-progress
  • a Windows Phone version may be created depending on how the webapp goes
  • JavaScript that makes this possible is a Gist here on GitHub.

Penn Philippine Association App

Following a lot of interest in the APO app's directory functionality from classmates, by the end of this summer I intend to generalize the patterns implicit in APO-DZ and build a generic directory module that college student organizations can adapt. The first fork shall be for my favorite collegiate Filipinos, the Penn Philippine Association.

Automated Theorem Deducers

As a big fan of recursion and functional programming, one of my hobbies has been working on modeling proof systems in Haskell. The source code will be open-sourced as things come to fruition.
